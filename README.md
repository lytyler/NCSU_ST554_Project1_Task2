# NCSU_ST554_Project1_Task2
Course Title: Analysis of Big Data    
Project 1 / Group Project    
Task 2 by Lanette Tyler    
(with feedback from project partners incorporated)

This project is an analysis of the air quality data set available from the UCI Machine Learning Repository. 

This portion of the project provides an extensive exploratory data analysis to understand the relationships between the true C6H6 values and the sensor values, and between the true C6H6 values and the weather variables.
